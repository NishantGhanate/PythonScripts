# Optical Character Recognition


### Installation



Install the dependencies for python 3.+  x64

```sh
$ pip install pytesseract
$ pip install opencv-contrib-python
$ pip install PyQt5
$ pip install PyQt5-tools
$ pip install pywin32
$ pip install pyinstaller 
```


### Download tesseract:
